## Hey, welcome 👋

This is the Skills API — a lightweight REST API built with Node.js, Express, and MongoDB. It was built to solve one specific problem: keeping track of which infrastructure skills your team members have, and making that data easy to query and manage.

There's no bloat here. You get a skill catalogue you can search, a way to record which skills belong to which user, and a protected admin endpoint to keep an eye on the numbers.

## How skills are structured

Skills live in a three-level hierarchy. Think of it like folders inside folders:

<CardGroup cols={2}>
  <Card title="MasterSkill" icon="layer-group">
    The broadest category — things like *Cloud*, *Networking*, or *DevOps*. Everything rolls up to one of these.
  </Card>
  <Card title="SubSkill" icon="sitemap">
    A more specific grouping inside a master skill. For example, *AWS* and *Azure* both live under *Cloud*.
  </Card>
  <Card title="Skill" icon="tag">
    The actual thing a person knows — *EC2*, *Lambda*, *Terraform*, etc. This is what users select.
  </Card>
  <Card title="SelectedSkill" icon="circle-check">
    A record that says "this user has this skill." One user, one skill — no duplicates allowed.
  </Card>
</CardGroup>

## Base URL

```
http://localhost:5000
```
