## What is the Skills API?

The **Skills API** is a REST API for managing infrastructure skills across your organisation. It lets you search a master skill catalogue, track which skills users have selected, and administer the catalogue through a protected admin interface.

## Data Model

Skills are organised in a three-level hierarchy:

<CardGroup cols={2}>
  <Card title="MasterSkill" icon="layer-group">
    Top-level skill domain, e.g. *Cloud*, *Networking*, *DevOps*. Everything rolls up to a master skill.
  </Card>
  <Card title="SubSkill" icon="sitemap">
    A grouping within a master skill, e.g. *AWS*, *Azure* under *Cloud*.
  </Card>
  <Card title="Skill" icon="tag">
    A concrete, leaf-level skill a user can select, e.g. *EC2*, *Lambda*, *S3*.
  </Card>
  <Card title="SelectedSkill" icon="circle-check">
    Links a **user** to a **skill** they have claimed. Each user can select a given skill only once.
  </Card>
</CardGroup>
