## Prerequisites

- [Node.js](https://nodejs.org/) v18+
- [MongoDB](https://www.mongodb.com/) running locally, or a connection string from [MongoDB Atlas](https://www.mongodb.com/atlas)
- npm

## 1. Clone the repository

```bash
git clone https://github.com/your-org/skills-api.git
cd skills-api
```

## 2. Install dependencies

```bash
npm install
```

## 3. Configure environment variables

Create a `.env` file in the project root:

```bash .env
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/skills_db
ADMIN_API_KEY=supersecretadminkey123
```

<Warning>
  Never commit your `.env` file to version control. Make sure it is listed in `.gitignore`.
</Warning>

## 4. Start the server

```bash
npm start
```

You should see:

```
MongoDB connected
Server running on port 5000
```

## 5. Make your first request

```bash
curl "http://localhost:5000/api/skills/search?keyword=docker&page=1"
```

Expected response:

```json
{
  "data": [
    {
      "skill_id": "sk-001",
      "Skill": "Docker",
      "sub_skill_id": "sub-001",
      "master_skill_id": "ms-001"
    }
  ],
  "page": 1,
  "limit": 10,
  "count": 1
}
```
