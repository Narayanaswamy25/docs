---
title: Delete Selected Skill
api: DELETE /api/selected-skills/{selected_skill_id}
description: "Remove a skill from a user's profile"
---

## Overview

Permanently removes a selected skill record by its UUID. This action is **irreversible**.  
Public endpoint — no authentication required.

---

## Path Parameters

<ParamField path="selected_skill_id" type="string" required>
  The UUID of the **selection record** to delete.  
  This is the `selected_skill_id` returned when the skill was added — **not** the `skill_id`.
</ParamField>

---

## Response

<ResponseField name="success" type="boolean">`true` when the record was deleted</ResponseField>
<ResponseField name="message" type="string">Status message</ResponseField>

---

## Examples

<CodeGroup>

```bash cURL
curl -X DELETE \
  "http://localhost:5000/api/selected-skills/a1b2c3d4-e5f6-7890-abcd-ef1234567890"
```

```js JavaScript
const res = await fetch(
  "http://localhost:5000/api/selected-skills/a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  { method: "DELETE" }
);
const data = await res.json();
```

```python Python
import requests
res = requests.delete(
  "http://localhost:5000/api/selected-skills/a1b2c3d4-e5f6-7890-abcd-ef1234567890"
)
print(res.json())
```

</CodeGroup>

<ResponseExample>
```json 200
{
  "success": true,
  "message": "Selected skill deleted successfully"
}
```

```json 404 - Not Found
{
  "success": false,
  "message": "Selected skill not found"
}
```
</ResponseExample>

---

<Warning>
  **Use `selected_skill_id`, not `skill_id`.**  
  The path parameter must be the UUID assigned to the *selection record* (returned from the `POST` response), not the skill's own ID.
</Warning>
